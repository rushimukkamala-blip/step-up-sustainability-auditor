Step Up — Unit Cost & Program Sustainability Auditor
A free, interactive web dashboard that transforms Step Up's eight years of real financial data into actionable program insights. Built for nonprofit finance teams operating under GAAP and 2 CFR Part 200 federal compliance requirements.
The tool answers three questions every nonprofit finance leader needs to address: what does it actually cost to serve one participant, are our programs financially sustainable, and what happens to service delivery if a grant gets cut?
Across four interactive tabs, users can explore an eight-year financial overview with live charts, a unit cost breakdown showing cost per student and cost per job placement by program, a scenario toggle with a drag slider that instantly models any funding cut from 0–50% and shows exactly how many students, counseling hours, and placements would be lost, and a sustainability audit with eight key indicators rated against sector benchmarks.
The 2023 data tells an urgent story: a ($305,774) operating deficit, grant dependency at 84.6%, and a program expense ratio that has slipped below the 85% sector standard. The tool surfaces these risks clearly and keeps them visible for leadership every time they open it.
It runs as a single HTML file — no installation, no login, no backend. It can be opened locally in any browser or deployed as a live public URL via GitHub Pages in under ten minutes.
